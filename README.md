## TP Back

### Fonctionnalités

- Connexion/Inscription utilisateur

### Utilisation

- Installer les dépendances avec `npm install`
- Builder le projet avec `npm run build`
- Et enfin lancer le avec `npm start`

### Développement

- Installer les dépendances avec `npm install`
- Lancer le projet en mode développement avec `npm run dev`
